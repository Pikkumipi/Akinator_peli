# Akinator_peli
Täällä on kaksi versiota akinator pelistä. Gemini LLM ja toinen Ollama.


OLLAMAN OHJEET

Koneella pitää olla asennettuna Ollama desktop versio ja siinä LLM versio qwen2.5:7b-instruct.

Tarkista ollama versio
ollama --version

Asenna kielimalli
ollama pull qwen2.5:7b-instruct

Aktivoi virtuaali ympäristö 
venv\Scripts\activate

lataa openai kirjasto
pip install openai



GEMINI OHJEET

Aktivoi virtuaali ympäristö (ei tarvitse jos paketit on asennettu onnistuneesti paikallisesti)
venv\Scripts\activate

lataa genai kirjasto 
pip install google-genai
